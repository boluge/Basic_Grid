Basic Grid

La grille

Télécharger

Basic Grid est un grille RWD en sass. Cette grille est mobile first. Elle est par defaut découpé en 12 colonnes, il est possible de choisir sa largeur, le nombre de colonnes, mais aussi la largeur de la goutière. Pour savoir comment porcéder, regardez le chapitre Configuration

L'utilisation

<!-- Classe Container -->
<div class="container">

	<!-- Classe Row -->
	<div class="row">
		<div class="col-6">
			...
		</div>
		<div class="col-6">
			...
		</div>
	</div>
</div>
<!-- Taille d'une colonne sur écran très petit -->
<div class="col-6">...</div>

<!-- Taille d'une colonne sur écran petit -->
<div class="col-s-6">...</div>

<!-- Taille d'une colonne sur écran moyen -->
<div class="col-m-6">...</div>

<!-- Taille d'une colonne sur écran large -->
<div class="col-l-6">...</div>
