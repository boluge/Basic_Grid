Basic Grid

Grille RWD

Télécharger

Basic Grid est un grille RWD en sass. Cette grille est mobile first. Elle est par defaut découpé en 12 colonnes, il est possible de choisir sa largeur, le nombre de colonnes, mais aussi la largeur de la goutière. Pour savoir comment porcéder, regardez le chapitre Configuration

L'utilisation

<!-- Classe Container -->
<div class="container">

	<!-- Classe Row -->
	<div class="row">
		<div class="grid-6">
			...
		</div>
		<div class="grid-6">
			...
		</div>
	</div>
</div>
<!-- Taille d'une colonne sur écran très petit -->
<div class="grid-6">...</div>

<!-- Taille d'une colonne sur écran petit -->
<div class="grid-s-6">...</div>

<!-- Taille d'une colonne sur écran moyen -->
<div class="grid-m-6">...</div>

<!-- Taille d'une colonne sur écran large -->
<div class="grid-l-6">...</div>

La Configuration

Les Variables

Vous pouvez modifier la grille en définissant les variables ci contre, avant d'importer grid.scss

/* Nb de colonnes */
$grid_columns: 12;
/* Marges entre colonnes */
$grid_gutter: 20px;

/* Largeur du container */
$container_extrasmall: 100%;
$container_small: 100%;
$container_medium: 80%;
$container_large: 1040px;

/* Tailles pour les Media Queries */
$grid_small: 720px;
$grid_medium: 960px;
$grid_large: 1040px;
